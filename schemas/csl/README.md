# Continuum Specification Language (CSL)

CSL is the declarative language used to define every object inside Continuum.

CSL does not describe implementation.

CSL describes intent, structure, authority, relationships, and responsibilities.

Everything inside Continuum is represented as a CSL document.

## Design Principles

- Human readable
- Machine executable
- Version controlled
- Domain agnostic
- Extensible
- Strongly structured
- Composable
- Deterministic

- Organization
